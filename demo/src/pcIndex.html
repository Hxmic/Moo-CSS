# 介绍

Moo-CSS (Moo, [mu:]哞~)是结合BEM、SMACSS、MCSS、OOCSS等写法思想总结而成的一套CSS写法规则，适用于大中小型C端项目样式开发，旨在提高开发和维护效率。

Moo-CSS有个大胆的想法： 统一CSS写法规范。
